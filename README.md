# Provincial Intelligent Safety | Limpopo Guardian

An AI-powered nervous system for public safety in Limpopo. This platform connects citizens, officers, and emergency responders through real-time intelligence and voice-activated assistance.

## Key Features

- **Smart SOS Orb**: A voice-activated and touch-sensitive emergency trigger with integrated AI guidance.
- **Voice Guardian**: "Always-On" speech recognition for rapid threat detection (Police, Ambulance, Fire).
- **Dual-Camera Protection**: Continuous recording of both the road and vehicle cabin during journeys.
- **Predictive Risk Analysis**: Real-time AI scanning of local crime data and traffic for proactive advice.
- **Medical & Guardian Profile**: Encrypted storage of blood type, allergies, and emergency contacts.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **AI**: Google Genkit with Gemini 2.5 Flash
- **Database/Auth**: Firebase Firestore & Firebase Auth
- **UI**: Tailwind CSS, ShadCN UI, Framer Motion
- **Icons**: Lucide React
